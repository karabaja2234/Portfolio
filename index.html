VueJS & Firebase Online Shop (newer project)

Fully working online shop made using VueJS, JavaScript, Firebase and other modern-day technologies. This shop features implementation of authentication for the users, realtime database for the products, orders and user information, as well as responsive design of every component included. Every user has an opportunity to make a profile, log in, put as many products as they want in the cart and order the products from the cart by entering their personal information at the checkout page. Each order is stored to database and the admin receives every one of them. Admin has full control over adding, deleting and updating products, orders and users. The shop also features the info page, where users can find an answer to some frequently asked questions.
